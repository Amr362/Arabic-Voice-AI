Build a production-ready Arabic AI Voice SaaS platform similar to ElevenLabs, optimized for Egyptian and Arabic users.

Tech Stack:
- Next.js 15
- JavaScript only (NO TypeScript)
- Tailwind CSS
- Supabase (Auth + Database + Storage)
- Railway deployment compatible
- Edge-TTS as default free voice engine
- Hugging Face XTTS-v2 API for premium AI voice cloning
- Responsive modern UI
- Dark mode support

Project Name:
ArabVoice AI

Main Goal:
Users can generate Arabic AI voices, clone voices, create TikTok voiceovers, and download audio files.

- Accept:

  - speaker audio upload
- Return WAV audio
- Loading states
- Error handling

IMPORTANT:
DO NOT run XTTS locally.
